About E-Vote Bank

Secure • Transparent • Fast

E-Vote Bank is an e-voting system designed to conduct elections digitally with controlled access, fair session handling, and privacy-safe vote visibility.

  • Admin starts and ends the voting session.
  • Voters can only vote and access restricted actions.
  • Admin can see total counts but not individual vote choices.
  • Results become visible only after voting ends.

How It Works

Admin controlled flow • Voter privacy protected

1

Admin Login

Admin logs in to control the election session.

2

Start Voting

Admin starts voting; system switches to LIVE state.

3

Voter Login & Vote

Voter logs in and casts a single vote. Other data stays restricted.

4

Counting (Privacy Safe)

Votes are counted as totals. Individual vote choices are not shown.

5

End Voting

Admin ends the session. Voting stops immediately.

6

Show Results

Results become visible only after voting ends.

Note: Admin can view totals and session status, but cannot see who voted for whom.

Security & Privacy

Protecting voters, ensuring fair counting

Role-based Access

Admin manages session; voters only cast votes.

Vote Privacy

Individual vote choices are hidden; only totals are shown.

Controlled Results

Results button stays disabled until voting is ended by admin.

Session Integrity

Voting can only occur during LIVE mode, preventing unauthorized votes.

Safe User Experience

Privacy guarantee: Admin can never view individual votes — only total counts.

Voting Guidelines

Please follow instructions carefully during voting

✔ Login Carefully

Enter correct voter credentials to access voting panel.

✔ One Vote Only

Each voter can cast vote only once.

✔ Check Before Submit

Verify selected party before confirming vote.

✔ Maintain Privacy

Do not share your vote or login details with anyone.

✖ Avoid Refresh

Do not refresh page while voting is in process.

✖ Unauthorized Access

Only registered voters can participate in voting.

Important: Voting results will be visible only after admin ends voting session.
